# SAR-INVENTORY
Sistema Web de Gestión Integral de Inventario Operativo para el Grupo de Búsqueda y Rescate (SAR).

## Tecnologías
- Backend: Python (Flask, SQLAlchemy)
- Frontend: HTML5, CSS3, Bootstrap 5
- Base de Datos: SQLite

## Ejecución
1. `pip install -r requirements.txt`
2. `python app.py`
